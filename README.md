Данный тг бот создан в качестве пет-проекта
Он предназначен для того, чтобы вносить в бд игроков встречающихся в катках в видеоиграх
Это может быть полезно, если в матчах попадаются одни и те же игроки, но ты ничего о них уже не помнишь, кроме никнеймов
Для работы необходимо создать .env файл, куда записать необходимые параметры

Телеграм токен: "TOKEN=*" и телеграм id администраторов(доп кнопки) "ADMINS=*"
